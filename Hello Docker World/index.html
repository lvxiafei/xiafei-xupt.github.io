Hello Docker World

Jan 11, 2018Hits

安装Docker命令：

#wget -qO- https://get.docker.com | sh

允许非root用户：

#usermod -aG docker xxx //把xxx用户添加到docker用户组中

拉docker镜像：

#docker pull nginx

查看已安装docker镜像：

#docker images

启动nginx服务

#docker run -p 8080:80 -d nginx

列出运行中的容器

#docker ps

编辑html文件，添加Hello Docker World

#vim index.html

拷贝文件：

#docker cp index.html 容器id://usr/share/nginx/html

查看主页：

localhost:8080
